Route:


Creating

  • Double click in empty space to create a new node
  • Shift click a node and drag to another node to create a link

Modifying

  • Click a node and drag to move
  • Double click cost text to change link cost

Deleting

  • Ctrl-click link cost labels to delete the link
  • Ctrl-click a node or its label to delete it
